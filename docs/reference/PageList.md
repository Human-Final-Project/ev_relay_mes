## 필수

### 1. LoginPage.js

- MES 사용자 로그인 화면

**화면 구성**

- 아이디 입력
- 비밀번호 입력
- 로그인 버튼
- 로그인 실패 메시지

---

### 2. DashboardPage.js

- 메인 화면

**화면 구성**

- 오늘 작업지시 수
- 진행 중 LOT 수
- 오늘 생산 OK 수량
- 오늘 NG 수량
- 가동 중 설비 수
- 발생 중 알람 수
- 공정별 OK/NG 막대 그래프
- 최근 생산 로그 테이블
- 설비 상태 카드

---

### 3. WorkOrderPage.js

- 작업 지시 생성/조회/상태 확인

**화면 구성**

- 작업지시 목록
- 작업지시 번호
- 생산 품목
- 목표 수량
- 상태
- 계획 시작일 / 종료일
- 작업지시 생성 버튼

**기능**

- 작업지시 조회
- 작업지시 생성
- 작업지시 상태 변경
- LOT 생성으로 연결

---

### 4. LotPage.js

**화면 구성**

- 작업지시 번호
- 품목
- 투입 수량
- OK 수량
- NG 수량
- 현재 공정
- 상태
- 시작일 / 완료일

모달이나 별도 영역 사용?

- LOT 기본 정보
- 공정별 생산 로그

- 불량 이력
